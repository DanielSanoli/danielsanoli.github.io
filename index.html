Projeto principal

Ecossistema PorygonZ — Microsserviços Pokémon

Conjunto de microsserviços inspirado no universo Pokémon, construído para demonstrar arquitetura distribuída, comunicação entre serviços, API Gateway, cache, persistência, integração com APIs externas, infraestrutura local com Docker Compose e evolução para recursos de IA.

O projeto é composto por serviços independentes em Java/Spring Boot e Python/FastAPI, organizados por responsabilidade e orquestrados por um repositório de infraestrutura.

Java Spring Boot Spring Cloud Gateway Python FastAPI Docker Compose PostgreSQL Redis Microsserviços
Gateway

porygonz-gateway

API Gateway responsável por centralizar o contrato público do ecossistema, roteando chamadas HTTP para os microserviços internos de Pokedex, comparação de tipos e fusões com IA.

  • Rotas versionadas em /api/v1/**
  • Centralização de CORS
  • Roteamento para serviços internos
  • Spring Cloud Gateway com WebFlux
Java Spring Boot Spring Cloud Gateway WebFlux
